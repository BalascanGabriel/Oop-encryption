Utilitarul implementeaza doua engine-uri de criptare(XOR si shift). Citeste datele dintr-un fisier de intrare dat 
ca argument  si are nevoie de numarul de biti cu care sa se shifteze pentru engine-ul de Shift si de o cheie(un sir
de caractere) pentru engine-ul de XOR.
La final va afisa in consola rezultatele si inainte de criptare si dupa ce textul a fost criptat.
